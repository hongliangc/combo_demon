# 2D TileMap 地图制作参考

> **Production Skill 已迁移至**: `.claude/skills/tilemap-level-design/SKILL.md`
>
> 该 Skill 会在讨论关卡/地图/TileMap 时自动触发，无需手动复制 prompt。

---

## 快速使用

直接对 Claude 说：

```
生成一个关卡:
  theme: forest
  size: 60x20
  difficulty: medium
  features: slopes, 1 secret area, water hazard
```

Claude 会按照 Skill 中定义的 Pipeline 自动执行：解析需求 → 地形骨架 → 平台 → 坡道 → 危险区 → Auto-tile → 装饰 → 验证 → 输出 GDScript。

---

## 项目层级结构速查

```
TileMap (Node2D)
├── BackRock   — 背景装饰（无碰撞）
├── Ground     — 主地形+平台（碰撞层1 "World"）
├── Water_layer1 — 水面/液体
├── Tree       — 大型植被
└── Grass      — 小型草地装饰（无碰撞）
```

- Tile Size: 16×16 px
- TileSet Atlas: `res://Assets/Art/HighForest/Assets/Tiles.png`
- Terrain auto-tiling: terrain_set 0, terrain 0

## 核心平台跳跃约束

| 约束 | Tiles | Pixels |
|------|:-----:|:------:|
| 最大水平跳跃间距 | 5 | 80 |
| 最大垂直跳跃高度 | 3 | 48 |
| 最小平台宽度 | 2 | 32 |
| 安全着陆区 | 3+ | 48+ |

## Skill 优化要点（相比旧版）

| 旧版问题 | 新版改进 |
|----------|---------|
| 两段内容重复（17节 + 13节） | 合并为单一无冗余 Skill |
| 不在 `.claude/skills/` 中 | 正确的 Skill 格式，自动触发 |

| 忽略项目实际结构 | 绑定项目层级（BackRock/Ground/Grass）、物理层、tileset |

| 无验证标准 | 强制 Validation Checklist + 报告输出 |
