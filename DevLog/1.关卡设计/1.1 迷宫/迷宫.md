# 二、平台迷宫结构类型

横版迷宫通常由 **三种结构组合**：

### 1 平台跳跃区

```text
   ███

        ███

 ███         ███
```

用途：

* 练习跳跃
* 控制节奏

---

### 2 垂直迷宫区

```text
     ███
  ███
       ███
  ███
       ███
```

玩家：

* 上下探索
* 找正确路径

---

### 3 掉落捷径

```text
██████████████

       ↓

   ███

       ↓

S
```

掉下来可以 **快速返回起点**

# 六、平台迷宫自动生成算法

关键限制：

```
最大跳跃高度 = 4 tiles
最大跳跃距离 = 5 tiles
```

生成平台必须满足：

```
|y2 - y1| <= jump_height
|x2 - x1| <= jump_distance
```

# 七、Godot 平台生成示例代码

示例 **GDScript**

```gdscript
extends Node2D

const LEVEL_WIDTH = 60
const LEVEL_HEIGHT = 25

const PLATFORM_COUNT = 35

@onready var tilemap = $TileMapLayer

func _ready():
    randomize()
    generate_level()

func generate_level():

    var last_pos = Vector2i(3, 20)

    for i in PLATFORM_COUNT:

        var x = clamp(last_pos.x + randi_range(-5,5), 2, LEVEL_WIDTH-2)
        var y = clamp(last_pos.y + randi_range(-3,3), 3, LEVEL_HEIGHT-3)

        var length = randi_range(2,5)

        for j in length:
            tilemap.set_cell(Vector2i(x+j, y), 0, Vector2i(1,0))

        last_pos = Vector2i(x,y)
```

生成效果类似：

```
   ███
        ███
  ███
           ███
      ███
```

保证：

* 基本可跳
* 有随机性

# 八、专业游戏常用结构（推荐）

横版迷宫通常是 **Room拼接系统**：

```
Room A → Room B
   ↓        ↓
Room C → Room D
```

地图：

```
      [Room]
         |
[Room]-[Room]-[Room]
         |
      [Room]
```

每个 Room 内是平台关卡。

优点：

* 地图更大
* 更容易设计
* 可重复利用

# 十、横版迷宫设计黄金规则（非常重要）

1️⃣ 平台高度差 ≤ **最大跳跃高度**

2️⃣ 平台距离 ≤ **最大跳跃距离**

3️⃣ 每 **10~15 tiles** 必须有落脚点

4️⃣ 避免连续盲跳

5️⃣ 设计 **回退路线**
